## tictactoe

**tictactoe** is an online multiplayer tic-tac-toe lobby. Players can create or join real time tic-tac-toe games driven by websockets. I created this project as an exploration of a number of technologies including React.js, Python FastAPI, and Websockets. 

The project is live on an [AWS EC2 instance](http://ec2-35-89-77-105.us-west-2.compute.amazonaws.com:3000/). You can try it out by playing against yourself in two different web browser tabs. 

See more information including engineering design thoughts on the [github](https://github.com/vayfarer/tictactoe/) readme. 

## CS 493 Cloud Application Development: Portfolio Project

The course project for cloud application development demonstrates REST API, Auth0 user account creation and user authentication for protected resources. Users may submit API requests to perform CRUD operations on boats and slips at a simulated harbor. 

One of my criticisms for CS 493 is that the course material focuses deeply on API development that happens to be performed on the google cloud platform. It only lightly touched cloud topics like containerizing and available cloud services.

## CS 344 Operating Systems: smallsh

Small shell (smallsh) is considered one of the more difficult assignments in the entire CS curriculum. Similar to other unix shell programs, smallsh is an interactive command-line interpreter and can be used to execute system commands. Some major features of smallsh include parameter expansion, executing non-built-in commands via the linux system call EXEC(3), redirection operators, and running commands in the background. 

## 2012 Northwestern Design Competition

Lyndon Sapozhnik and I formed a team to compete in the 2012 Northwestern Autonomous Robot Design Competition. Design Competition is a complete engineering challenge integrating several engineering disciplines including electrical, mechanical, and software. The objective of the 2012 competition was to drive and pick up foam cubes on a 6 ft x 12 ft field and then deposit the cubes in a mobile scoring zone. 

Our robot named Esteban [won the competition](https://www.mccormick.northwestern.edu/news/articles/2012/05/2012-design-competition-winners.html)! 

Lyndon has posted a very well documented readme and Esteban's algorithm on a [github repo](https://github.com/lsapoz/esteban).
